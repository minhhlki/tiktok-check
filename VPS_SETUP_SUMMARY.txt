                            CÁCH 1: TỰ ĐỘNG (NHANH NHẤT)
═══════════════════════════════════════════════════════════════════════════

1️⃣ SSH vào VPS:
   ssh username@your-vps-ip

2️⃣ Chạy 3 lệnh này:

   git clone https://github.com/minhhlki/tiktok-check.git
   cd tiktok-check
   chmod +x setup_ubuntu.sh && ./setup_ubuntu.sh

3️⃣ Chạy tool:

   source venv/bin/activate
   python tiktok_counter_improved.py https://www.tiktok.com/@username --save

✅ XONG! Đơn giản vậy thôi!

                    SAU KHI CÀI XONG - SỬ DỤNG THẾ NÀO?
═══════════════════════════════════════════════════════════════════════════

🔄 MỖI LẦN CHẠY (sau khi reboot hoặc mở terminal mới):

   cd ~/tiktok-check
   source venv/bin/activate

🚀 CHẠY TOOL:

   # Phiên bản improved (chính xác hơn - KHUYÊN DÙNG)
   python tiktok_counter_improved.py https://www.tiktok.com/@username --save

   # Phiên bản thường (nhiều options hơn)
   python tiktok_counter.py https://www.tiktok.com/@username --save-json --save-csv

   # Hoặc dùng helper script
   ./run_tiktok_improved.sh https://www.tiktok.com/@username --save

📥 DOWNLOAD KẾT QUẢ (từ máy local):

   # Download JSON
   scp username@vps-ip:~/tiktok-check/*.json ./

   # Download CSV
   scp username@vps-ip:~/tiktok-check/*.csv ./

   # Download tất cả
   scp username@vps-ip:~/tiktok-check/*.{json,csv} ./

                        TROUBLESHOOTING - LỖI THƯỜNG GẶP
═══════════════════════════════════════════════════════════════════════════

❌ Lỗi: "command not found"
   ✅ Chạy: source venv/bin/activate

❌ Lỗi: "Could not find browser"
   ✅ Chạy: playwright install chromium && playwright install-deps chromium

❌ Lỗi: "Permission denied"
   ✅ Chạy: chmod +x setup_ubuntu.sh
         hoặc: sudo chown -R $USER:$USER ~/tiktok-check

❌ Lỗi: "Out of memory"
   ✅ Tạo swap: sudo fallocate -l 2G /swapfile
                sudo chmod 600 /swapfile
                sudo mkswap /swapfile
                sudo swapon /swapfile

❌ Lỗi: "Timeout" khi scrape
   ✅ Tăng timeout trong code hoặc dùng VPN

                        CHẠY TỰ ĐỘNG ĐỊNH KỲ (OPTIONAL)

1️⃣ Tạo script:
   nano ~/tiktok-check/auto_run.sh

   Nội dung:
   #!/bin/bash
   cd ~/tiktok-check
   source venv/bin/activate
   python tiktok_counter_improved.py https://www.tiktok.com/@YOUR_CHANNEL --save

2️⃣ Cho phép execute:
   chmod +x ~/tiktok-check/auto_run.sh

3️⃣ Setup cron (chạy mỗi ngày 3h sáng):
   crontab -e
   
   Thêm dòng:
   0 3 * * * /home/username/tiktok-check/auto_run.sh >> /home/username/tiktok-check/auto.log 2>&1

═══════════════════════════════════════════════════════════════════════════
                              TIPS QUAN TRỌNG
═══════════════════════════════════════════════════════════════════════════

💡 Luôn nhớ activate venv:
   cd ~/tiktok-check && source venv/bin/activate

💡 Dùng phiên bản improved để có độ chính xác cao hơn:
   python tiktok_counter_improved.py ...

💡 Không chạy quá nhiều lần liên tục (tránh bị TikTok block):
   - Đợi ít nhất 5-10 phút giữa mỗi lần
   - Hoặc dùng VPN/proxy

💡 Monitor tài nguyên:
   free -h        # Check RAM
   df -h          # Check disk
   htop           # Check processes

💡 Chạy trong background (không bị ngắt khi logout):
   nohup python tiktok_counter_improved.py URL --save > output.log 2>&1 &
